# Naomi's Shave Ice Stand

A simple toddler-friendly React/Vite web game inspired by a Hawaiian shaved ice stand and mermaid play set.

## Play

```bash
npm install
npm run dev
```

Open the local URL Vite prints, usually `http://127.0.0.1:5173/`.

## Game Flow

- Match the customer's three picture requests: cup, ice flavor, topping.
- Click **Make It!** to serve the icee.
- A happy heart is added for a correct match.
- The next customer appears automatically.
- The reset button starts over.
